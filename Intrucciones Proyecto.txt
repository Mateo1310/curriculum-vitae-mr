en grupos de mínimo 2 maximo 4 se debe realizar por cada miembro una pagina web de sus  HOJAS DE VIDA (CV) con HTML, CSS, BOOTSTRAP, GIT de la cual se debe dividir de la siguiente forma:

1. cada miembro debe encargarse de una sección ejemplo: (menú, experiencia laboral, carrusel de imágenes, etc...)
2. de cada sección cada miembro debe requerir a los demás compañeros de trabajo los datos necesarios para crearla
3. cada miembro debe tener una rama propia en git y debe evidenciarse mínimo 10 commits en su rama personal desde su cuenta personal
4. el proyecto grupal se debe subir a GITLAB comoPRIVADO y debe estar compartido con los demás miembros del equipo
5. en la totalidad del proyecto deben existir mínimo 5 mezclas o conflictos
6. el proyecto de gitlab se debe subir al servidor NETLIFY
7. cada miembro debe descargar el proyecto individual tenerlo en otra carpeta borrar la carpeta oculta .git para volver a inicializar git y subir el proyecto a GITHUB
8. cada miembro debe colocar su proyecto publico y debe subirlo o implementarlo en GITHUB PAGES
9. el proyecto grupal en GITLAB debe tener un readme.md presentable y bonito
10. el proyecto individual en GITHUB debe tener un readme.md  presentable y bonito
11. el proyecto grupal debe tener una carpeta que se ignorara con varios archivos .pdf 
12. el proyecto debe verse bien tanto en html y css cumpliendo indentacion y etiquetas meta necesarias
